# Tours-and-Travels_Management_System

## Overview

Tours-and-Travels_Management_System is a PostgreSQL-based database management system for a travel booking platform. It manages users, vendors, hotels, transport services, holiday packages, bookings, payments, insurance, reviews, and cancellations using a normalized relational database.

The project includes the complete database schema, sample data, ER diagram, normalization proofs, and SQL queries for different types of users.

## Features

- User and Vendor Management
- Hotel Booking Management
- Transport Booking Management
- Holiday Package Management
- Booking and Payment Tracking
- Promo Code Support
- Travel Insurance Management
- Reviews and Ratings
- Booking Cancellation and Refund Management
- Analytical SQL Queries

## Setup

1. Create a PostgreSQL database.
2. Execute `DDL_scripts.txt` to create all tables.
3. Execute `Insert_scripts.txt` to insert sample data.
4. Run the queries from `Queries_Solutions.txt`.

## Files

- **DDL_scripts.txt** – Contains all table creation scripts.
- **Insert_scripts.txt** – Contains sample data for the database.
- **Queries_Solutions.txt** – Contains SQL queries for guests, travelers, and vendors.
- **ERD_Relational_Schema_and_Normalization_Proofs.pdf** – Includes the ER diagram, relational schema, and BCNF normalization proofs.
